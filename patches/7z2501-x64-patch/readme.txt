=> How it works?

1. Go to [7-zip](https://www.7-zip.org/) official website and download the 64-bit Windows x64 version installation file, 
   and install it to your computer.

2. Find and download the patch file for your favorite icon set (file icon or toolbar icon) below;

3. Copy the downloaded 7z.dll or 7zFM.exe file to the location where you installed 7-zip and replace the original file;
